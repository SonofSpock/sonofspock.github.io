Lists

HTML has three separate types of lists built into it.

Unordered Lists

These lists use bullet points because there's no particular order associated with them. You can also nests a list inside of another list to make a hierarchy.

  • Bread
  • Milk
  • Eggs
  • Fruit
    • Apples
    • Oranges
    • Bananas
  • Coffee

Ordered Lists

Ordered lists, on the other hand, use numbers to indicate that a particular sequence matters for the items in the list. These lists can also be nested within one another. Thankfully, HTML takes care of the numbering automatically.

  1. Preheat oven to 450 deg
  2. Combine dry ingredients in a large mixing bowl
    1. Flour
    2. Baking Soda
    3. Salt
    4. Sugar
  3. Add wet ingredients, and mix until all lumps are gone
  4. Pour evenly into 9 in. baking pan, and bake for 30 mins.

Description Lists

This final type of list should be used when each element of the list has its own definition or description. For example, suppose you had a glossary or a list of terms.

Python
A dynamically typed, interpreted, scripting language for general purpose computer programming. It has recently become popular for web development and for data science and analytics.
JavaScript
Another dynamically typed, interpreted language mostly used for client-side web development. Recently, however, with the development of NodeJS, its popularity as a server-side language has increased.
R
This is also a dynamically typed, interpreted language. It is mostly used for data science and analytics. It is very popular among statisticians.

Built-in References

HTML offers a great set of tags for marking up blockquotes, inline quotes, abbreviations, and citations. The quote below is taken from Barrack Obama's inaugural address.

Our journey has never been one of short-cuts or settling for less. It has not been the path for the faint-hearted - for those who prefer leisure over work, or seek only the pleasures of riches and fame. Rather, it has been the risk-takers, the doers, the makers of things - some celebrated but more often men and women obscure in their labor, who have carried us up the long, rugged path towards prosperity and freedom.

That text is offset from the rest of the text on the page becaues it is wrapped in <blockquote> tags. But not all quotes are long enough or significant enough to warrant a blockquote. For these inline quotes we use the <q> tag. Both have a cite attribute.
