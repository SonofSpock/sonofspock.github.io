I'm currently working as a software developer at DealCloud on the Technical Services team. My work primarily deals with the data migration process for our clients as they transition from their current data solution to the DealCloud CRM. Each migration is unique, and so each solution has its own implementation. But to help some mundane data processing tasks, we have an internal web application known as the Data Tools for which I am primarily responsible. It is a full-stack solution built with Flask and ReactJS. Data processes are managed by an asynchronous task queue with Celery and Redis.

Prior to DealCloud, I worked for Skyfactor from 2017 to 2019 as a data analyst. In that role I was primarily for developing statistical models for student retention. I also developed some internal applications to automate aspects of our production process, and provided insights to our operations team.
